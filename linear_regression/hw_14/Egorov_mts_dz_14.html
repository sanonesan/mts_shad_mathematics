$$ F = \dfrac{(RSS_r - RSS_u) / q}{RSS_u / (n - k - 1)} $$ $$ F = \dfrac{(R^2_u - R^2_r) / q}{(1 - R^2_u) / (n - k - 1)} $$

: $$ RSS_r = RSS / df_r, \quad RSS_u = RSS / df_u, $$ $$ R^2_{r} = 1 - \dfrac{RSS / df_r}{TSS / df_t}, \quad R^2_{u} = 1 - \dfrac{RSS / df_u}{TSS / df_t}. $$

$F$- $$ F = \dfrac{(RSS_r - RSS_u) / q}{RSS_u / (n - k - 1)} = \dfrac{\left( (1 - R^2_r) TSS / df_t - (1 - R^2_u) TSS / df_t\right) / q}{\left( (1 - R^2_u) TSS / df_t \right) / (n - k - 1)} = \dfrac{(R^2_u - R^2_r) / q}{(1 - R^2_u) / (n - k - 1)}. $$
